Contributor & Authorship Guidelines:
We welcome all contributions to this project via GitHub issues and pull requests. Your ideas matter more than perfect pull requests. All contributions must follow our Guiding Principles and Code of Conduct.

Contributor Guidelines:
Contributions are welcome from researchers, experts, and community members. This participatory approach brings multiple perspectives into the research from the start and ensures peer review happens for all outputs, not just articles at the end.

Ways to contribute Research Assistance;
- Conduct experiments as students, researchers, or research assistants
- Submit pull requests with clear descriptions
- Follow the project style guide for formatting

Expertise:
- Provide feedback on results on github
- Create tools essential for carrying out research
- Provide data expertise and specialized support
- Review code, methods, or documentation

Community & Management:
- Manage community and project requirements
- Help with writing, editing, and documentation
- Support outreach and social media

Authorship Guidelines:
Authorship is based on substantial contribution to one or more of the following:
    
1. Conceptualizing the idea or designing the project
2. Acquisition, analysis, or interpretation of data
3. Drafting the work or reviewing it critically for important content
4. Final approval of the version to be published
5. Managing community/project requirements and agreements
6. Accountability for all aspects of the work

All authors must approve the final version.  
Author order: alphabetical by last name unless agreed otherwise.

Acknowledging Contributors

Contributors will be acknowledged in these ways:
- Footnotes or appendices in published papers
- A "Contributors" section on the project website or GitHub page
- A "Thank You" section in presentations or talks about the project
- Social media shoutouts for notable contributions

Types of contributions we acknowledge:
- Research assistance and data analysis
- Expertise, feedback, and tool creation
- Community management and support
- Writing and editing
- Data provision and curation
